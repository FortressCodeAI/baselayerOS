# Contributing to BaseLayerOS

Thank you for your interest in contributing to the BaseLayerOS public standard.

This repository defines the *open specification* for deterministic AI governance.
It does not contain the enterprise runtime or premium modules.

## How to Contribute

### 1. Propose Specification Improvements

You may propose changes to:

- schemas in `standard/schemas/`
- documentation in `docs/`
- examples in `standard/examples/`
- SDK interfaces in `sdk/python/`

Open a GitHub Issue describing:

- the problem
- the proposed change
- the rationale
- any backward‑compatibility considerations

### 2. Submit Pull Requests

Pull requests should:

- modify only public specification files
- include tests or examples when relevant
- maintain backward compatibility unless explicitly discussed

### 3. What You Cannot Contribute

This repository does *not* accept contributions to:

- the deterministic runtime engine
- compliance enforcement logic
- provenance or lifecycle engines
- premium or enterprise modules
- marketplace or revenue systems

Those components are proprietary.

## Code of Conduct

All contributors must follow the CODE_OF_CONDUCT.md.
